# godot-toon-lit
Simple Toon Lit Shader for Godot

What's Included in the project files:
- Example Scene
- Example Material
- The Shader itself

Current Features:
 - Rim Lighting (wip)
 - Specular Highlights
 - Toon Shading (obv)
 - Multiple Light support
